Project 1

Backend de Generador de Itinerarios para viajes con IA

Descripción: Este proyecto consistía en crear un sistema que generara itinerarios de viaje automáticamente basándose en las preferencias del usuario, utilizando Javascript. Se uso la API de gemini AI para que genere un itinerario y con fechas, localizaciones y duracion. Eso se guardo en un JSON para que se pueda usar con SQLite y agregar a la base de datos cada viaje. Asi se genero una api generadora de itinerarios. Se uso AWS para hostear el proyecto

Tecnologías: SQLite, Express.js, typescript, JS, AWS

Duración: 2 semanas

Link al código: https://github.com/AgusRodrigues/TP3_APP5.git

Aplicacion Fitness gamificada

Prototipo de una aplicación de fitness gamificada diseñada para motivar a los usuarios a alcanzar sus objetivos de salud y bienestar mediante un sistema de recompensas dinámico. La aplicación incluye logros y medallas coleccionables, así como un perfil personalizable. Los usuarios pueden elegir entre tres modalidades de entrenamiento adaptadas a sus metas, ya sea bajar de peso, tonificar el cuerpo o aumentar la fuerza.

Aplicacion Museo

Prototipo de una aplicación para visitas a museos, diseñada para facilitar la exploración y el aprendizaje. La aplicación incluye una guía interactiva con mapa para ubicar rápidamente las salas de interés, así como videos e imágenes de las exhibiciones para ayudar a los visitantes a descubrir y priorizar las muestras que más les atraen. Cada sala cuenta además con una vista previa que ofrece un adelanto de lo que se puede encontrar, enriqueciendo la experiencia de visita

Dieet

Prototipo de una aplicación de nutrición diseñada para ayudar a personas con restricciones alimenticias a encontrar y adquirir alimentos que se ajusten a sus necesidades específicas. La aplicación permite explorar tiendas con productos compatibles y realizar pedidos para envío a domicilio. También incluye una sección de recetas adaptadas para quienes necesitan orientación en la cocina debido a sus restricciones. Además, la aplicación muestra las tiendas cercanas que ofrecen alimentos alineados con el perfil alimenticio personalizado del usuario, asegurando una experiencia de búsqueda eficaz y centrada en sus preferencias.

NAVI

Prototipo de una aplicación web de tienda para juegos indie, diseñada para conectar a jugadores interesados en títulos independientes menos convencionales y a desarrolladores que buscan una alternativa de distribución fuera de plataformas como Steam o Epic Games. La tienda permite a los usuarios personalizar su perfil incluyendo las especificaciones de su computadora, asegurando que solo se muestren juegos compatibles con su sistema. Esta funcionalidad brinda una experiencia optimizada tanto para consumidores como para creadores de contenido indie.
